#  基物实验计算器

一款基于给台场静马打电话的北航基物实验小工具（bushi）

非常不正经的介绍：

基物实验计算器

面向大二基物实验所必须又恶心的一系列计算。有了它，就不用每周痛一次

功能是算回归和不确定度，还有相关的一系列数据。
